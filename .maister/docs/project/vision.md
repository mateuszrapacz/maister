# Vision

Maister is a safe, auditable, resumable SDLC plugin distributed across multiple AI hosts. Its source of truth is a portable common layer with explicit native overlays and a target-aware transactional installer.

The project optimizes for:

- one owner for portable behavior;
- visible, versioned host contracts;
- immutable source provenance;
- byte-exact recovery of user state;
- evidence that distinguishes unsupported native runtime from a passing capability;
- small, behavior-focused tests and reproducible releases.

The supported targets are Codex, Cursor, and Kiro CLI. Legacy host support, marketplace projections, and committed generated trees were migration-era inputs; they are removed from the supported topology and retained only where explicitly identified as historical parity context.
